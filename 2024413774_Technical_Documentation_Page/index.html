Introduction

CSS Grid Layout is a highly powerful two-dimensional layout system designed specifically for the web. It allows developers to create complex grid structures with rows and columns smoothly without relying heavily on floats or positioning.

Unlike Flexbox, which is primarily one-dimensional (dealing with either a single row or column at a time), CSS Grid works effortlessly across both dimensions simultaneously.

To initialize a grid, you simply apply the following property to a parent element:

display: grid;
Grid Container

The element that has display: grid applied to it becomes the direct grid container. All immediate children of this element automatically become grid items.

Once you establish a container, you can define its structural columns and rows using explicit sizing parameters. This container manages the foundational structural layout properties for its direct descendants.

Here are basic rules for grid containers:

  • They establish a new grid formatting context.
  • They can utilize alignment properties like justify-items.
  • They pass structural rules down to direct children.
Grid Tracks

Columns and rows are collectively referred to as grid tracks. You define these tracks on your grid container using specific properties that dictate their sizing parameters.

For instance, to create three columns of equal width, you can use the grid-template-columns property like this:

grid-template-columns: 1fr 1fr 1fr;

The 1fr unit stands for "fractional unit", which automatically calculates a fraction of the available free space inside the container.

Placing Items

By default, grid items position themselves automatically according to source order. However, you can explicitly control where an item starts and ends using line numbers.

Grid lines are the numbered lines that divide the columns and rows. You can target specific lines using item properties:

grid-column: 1 / 3;

This code tells a single item to span from the first vertical grid line up to the third vertical grid line, occupying two columns total.

  • Line numbers start at 1.
  • Negative numbers like -1 target lines from the end.
Advanced Layouts

For more intuitive and scalable setups, CSS Grid offers advanced features like grid template areas and auto-fitting techniques.

Using the repeat function alongside auto-fit allows you to create beautifully responsive layouts completely without media queries:

grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));

This single rule establishes as many 200px columns as the viewport can fit, instantly distributing remaining fractional workspace equally across items.
